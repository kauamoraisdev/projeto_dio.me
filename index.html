ㅤㅤㅤTAGS

Elas servem para informar ao navegador como formatar o texto, estruturar o conteúdo e definir a estrutura de uma página da web

Por exemplo:

  • < p > : Indica o início de um parágrafo.
  • < /p > : Indica o fim de um parágrafo.
  • < h1 > : deixa texto em Negrito e com uma fonte grande, ideal para se usar de cabeçario
  • < mark > : essa tag deixa marcado como um marca-texto no qual usamos na escola para marcar algo.
    • Essas foram algumas das várias de tags que existem.
